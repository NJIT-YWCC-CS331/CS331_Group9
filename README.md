# ✈️ Flight Ticket Booking System  
### *Web-Based Database Application built with PHP, Oracle, HTML/CSS, and XAMPP*

---

## Project Overview  
The **Flight Ticket Booking System** is a web-based application that integrates a fully normalized **Oracle database** with a **PHP backend**, and an **HTML/CSS frontend**. Passengers can register, log in, search flights, book tickets, and view their reservations.  
Administrators can access a dashboard to manage and review all users and reservations.

This project demonstrates complete end-to-end development using:

- **Oracle SQL** (remote school server)  
- **PHP with OCI8 driver (enabled OCI8 extension in PHP)**  
- **XAMPP / Apache Web Server**  
- **HTML/CSS** for the UI  

---

## User Functionalities  
- **User Registration**  
- **Login (Email + DOB authentication)**  
- **Profile Display**  
- **Search Flights**  
- **Reserve / Book Ticket**  
- **View My Tickets**  

---

## Admin Functionalities  
- **Admin Login**  
- **Admin Dashboard**  
- **View Registered Users**  
- **View All Reservations/Tickets**  

---

## Running the Application  

1. Install and launch **XAMPP** (Apache)
2. Ensure PHP has the **OCI8 extension** enabled to connect to Oracle
3. Place the project folder in:  C:\xampp\htdocs\TermProject
4. Start the Apache Server
5. Open Application in browser: http://localhost/TermProject/login.php
